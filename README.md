# Marcão Rastreamento

### :question: Sobre

Aplicação react para rastreamento de pacotes dos Correios Brasil.

#### Tecnologias usadas:

- [ReactJS](https://pt-br.reactjs.org/)
- [Styled-Components](https://styled-components.com/)
- [React Spring](https://react-spring.io/)
